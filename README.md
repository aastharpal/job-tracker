# 📌 Job Tracker Web Application  

A full-stack web application that helps users efficiently manage and track their job applications.  
The app provides a clean, responsive dashboard with dynamic UI updates and persistent data storage.  

---

## 🔗 Features  

- Add and manage job applications  
- Track application status (Applied, Interview, Offer, Rejected)  
- Edit and update existing entries  
- Delete unwanted records  
- Responsive and interactive UI with animations  
- Color-coded status indicators for quick insights  
- Persistent storage using MySQL  

---

## 🛠️ Technologies Used  

- HTML5  
- CSS3  
- JavaScript (ES6, Fetch API)  
- Node.js  
- Express.js  
- MySQL  
- mysql2  

---

## ⚙️ How It Works  

1. User enters job details through the interface  
2. Frontend sends requests using Fetch API  
3. Backend processes requests via RESTful APIs  
4. Data is stored and retrieved from MySQL  
5. UI updates dynamically without page reload
